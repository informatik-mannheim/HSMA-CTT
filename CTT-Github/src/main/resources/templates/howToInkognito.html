Inkognito im Firefox

Zum Anmelden von mehreren Personen müssen Sie lediglich einen neuen Inkognito-Tab auf ihrem Endgerät öffnen.

Drücken Sie lange, oder mit einem Rechtsklick auf den link und wählen Sie "Link in privatem Tab öffnen" aus, um direkt auf die Anleitung zum einloggen zu gelangen

Alternativ können Sie zuerst auf das Tab Symbol im Browser drücken, dann auf das Maskensymbol (Privates Surfen)

Inkognito im Chrome

Zum Anmelden von mehreren Personen müssen Sie lediglich einen neuen Inkognito-Tab auf ihrem Endgerät öffnen.

Drücken Sie lange, oder mit einem Rechtsklick auf den link und wählen Sie "Link in Inkognito-Fenster öffnen" aus, um direkt auf die Anleitung zum einloggen zu gelangen

Drücken Sie zuerst auf die drei Punkte, dann auf "Neuer Inkognito-Tab"

Inkognito im Safari

Zum Anmelden von mehreren Personen müssen Sie lediglich einen neuen Inkognito-Tab auf ihrem Endgerät öffnen.

Drücken Sie zuerst auf das Tab-Symbol unten rechts

Dann drücken Sie unten links auf Privat
